# RuLiLuRa-CharaSheet
ルリルラ・ノイシュタルト用キャラクターシート管理システム

# 準備
WSL で以下：
```
sudo apt update
sudo apt install -y curl
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

一度ターミナルを開き直してから：
```
nvm install 22
nvm use 22
node -v
npm -v
npm install
npm install lz-string
npm i -D vitest jsdom
```

# ローカルで起動
```
npm run dev
npm run test
```

# ルールブックの不備
- 多くのアイテムについて効果の記載がない
- 一部のフリーアイテムについてアイテム欄に記載がない。
    - "id": 34, "name": "偽造身分証明書"
    - "id": 35, "name": "ナベ・フライパン"
    - "id": 36, "name": "ダイス"
    - "id": 37, "name": "虫メガネ"
- 「必要スキルを満たしていない射撃武器の場合、攻撃は1回しかできません（弾薬の調達や、適切な弾込めが行えない）」
との記載があるが、命中率へのペナルティに加えてシナリオ中に武器を一度しか使用できない制約を付けるゲームバランス上の必然性が不明である。
- 「手持ち4連ロケットポッド」の攻撃回数が3回
- 「大型ロケットランチャー」「小型ロケットランチャー」は特に説明が無いが攻撃回数が4回
- 攻撃回数は同じにもかかわらず、「大型ロケットランチャー」は撃ち分け可能、「小型ロケットランチャー」は撃ち分け不能
- 「グレネード」の射程: 1*にかんしてアスタリスクの意味が説明されていない
- 二刀流には専用の英雄能力が必要との記載があるが、近接武器の二刀流に対応するスキルは存在しない。
- 「ノイズ発生機」-> ノイズ発生器
- 「歌術：奏甲戦闘補助の歌」ランク３Ｂの効果がランク３Ａの効果を包含している（選択の余地が無い）
